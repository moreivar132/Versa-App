# Banking Module

Este directorio contiene la implementación del módulo de importación bancaria manual.

## Documentación
Para ver la documentación detallada del módulo, visite:
[docs/MODULES/BANKING.md](../../../docs/MODULES/BANKING.md)

## Estructura
- `controllers/`: Manejo de peticiones.
- `services/`: Lógica de negocio (Importación, Procesamiento).
- `parsers/`: Estrategias de lectura de archivos (CSV, XLS).
- `routes/`: Endpoints de la API.
